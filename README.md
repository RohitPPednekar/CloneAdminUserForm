# AdminCreatedUserForm

A simple form for the user that admin will create and then will displayed to the user

Note (Make sure):-

NPM and Node is installed in your local machine,
Angular/cli is installed in your local machine,
To run the project follow steps:-

Download the clone of this repository,
Install all the node packages by command "npm install",
Run the project by executing command "ng serve",
Go on browser and type http://localhost:4201
